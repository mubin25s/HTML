Use Case Description

Use Case Sponsor Registration & Management
Goal To manage event sponsors, their tiers (Gold, Silver), and logo visibility.
Preconditions Event is created.
Success End Condition Sponsors are listed on the event page.
Failed End Condition Sponsor data not saved.
Primary Actors Event Organizer
Secondary Actors Sponsor (Entity)
Trigger "Add Sponsor" action.
Description / Main Success Scenario
StepAction
1Organizer accesses "Sponsors" section.
2Organizer creates sponsorship tiers (e.g., Gold - $5000).
3Organizer adds Sponsor Name, Logo, and Website URL.
4Organizer assigns Sponsor to a Tier.
5System formats and displays Sponsor Logos on the Event Landing Page.
Alternative Flows
StepBranching Action
3aLogo Resize: System auto-resizes large images to fit standardized slots.
Quality Requirements
StepRequirement
5Logos must differ in size based on Tier (Gold > Silver) automatically.
