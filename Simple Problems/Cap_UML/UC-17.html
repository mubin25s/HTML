Use Case Description

Use Case Document Upload
Goal To share resources (slides, brochures) with attendees.
Preconditions Organizer has file.
Success End Condition File hosted and downloadable.
Failed End Condition Upload fail.
Primary Actors Organizer
Secondary Actors Attendee (Consumer)
Trigger "Upload Resource".
Description / Main Success Scenario
StepAction
1Organizer selects event session.
2Organizer drags and drops file (e.g., "Presentation.pdf").
3System scans for viruses.
4System uploads to cloud storage.
5System generates download link on Event Page.
Alternative Flows
StepBranching Action
3aVirus Detected: System rejects file. Warns User.
Quality Requirements
StepRequirement
4Upload progress bar shown for files > 1MB.
