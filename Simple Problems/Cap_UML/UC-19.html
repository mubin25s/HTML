Use Case Description

Use Case Event Closure & Logout
Goal To formally close an event and securely terminate the user session.
Preconditions Event date passed OR User wants to leave.
Success End Condition Event state = "Closed" OR User session destroyed.
Failed End Condition None.
Primary Actors Organizer / Any User
Secondary Actors System
Trigger "Close Event" or "Logout".
Description / Main Success Scenario
StepAction
1(Flow A: Close Event) Organizer clicks "Close Event".
2System archives event data.
3System restricts further registrations/edits.
4(Flow B: Logout) User clicks "Logout".
5System invalidates session token.
6System redirects to Homepage.
Alternative Flows
StepBranching Action
4aAuto-Logout: System detects inactivity > 30mins. Logs out user automatically.
Quality Requirements
StepRequirement
5Session data must be completely cleared from browser cache/storage directives.
